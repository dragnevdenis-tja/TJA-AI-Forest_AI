# Migration Notes - Forest Audio AI Refactoring

The project has been restructured from a script-based prototype into a modular, production-ready system. Existing files have been moved into a structured `backend/` and `frontend/` layout to improve maintainability and scalability. The `backend/` directory now contains dedicated subdirectories for APIs, services, ML models, data pipelines, and utilities. Foundational infrastructure has been added, including a centralized `config.yaml` for configuration management, and utility modules for logging (`logger.py`), deterministic seeding (`seed.py`), and configuration loading (`config.py`). This restructuring ensures a clear separation of concerns, makes the codebase easier to test, and provides a robust foundation for future ML and IoT integrations.
